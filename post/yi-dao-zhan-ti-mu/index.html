一道栈题目

2022-07-21

练习：栈和排序

题目描述

  • 给你一个1->n的排列和一个栈，入栈顺序给定。

  • 你要在不打乱入栈顺序的情况下，对数组进行从大到小排序。

  • 当无法完全排序时，请输出字典序最大的出栈序列。

输入描述

  • 第一行一个数n。

  • 第二行n个数，表示入栈的顺序，用空格隔开，结尾无空格。

输出描述

输出一行n个数表示答案，用空格隔开，结尾无空格。

输入

5
2 1 5 3 4

输出

5 4 3 1 2

样例说明

2入栈；1入栈；5入栈；5出栈；3入栈；4入栈；4出栈；3出栈；1出栈；2出栈

友情提醒

如果对这道题没有思路，还是强烈建议自己想一下，如果真没有思路，建议出门右转打个河道蟹再回来。

题目分析

这里的排序不是真的排序，只需要按照字典序最大输出即可。所以对于一个排列，我们需要把n放在第一位，第二个我们去找n-1，如果n-1在n在序列中位置之后出现，我们就应该等到n-1出现后输出n-1，如果n-1已经入栈，如果不是栈顶，那么我们是没办法立即取出的，这时候第二个输出的应该不是n-1了，我们应该去找除了栈中最大的数字，这个是很不好找的，我发现了一个布尔滑梯（随便起的），就是首先有一个值s等于不在栈中的最大值，开始时是n，每输出一次都会减少，减少直到栈中没有这个值，那么这个值就是非栈中的最大值s。我们找到一个这样的值s，那么输出，然后我们取出栈顶的值top，栈顶的值top我们是可以马上操作的，如果栈顶的值top大于s，那么我们最应该输出栈顶的值top，然后再取出栈顶的值top继续这一操作。

嗯，就酱。

#include <bits/stdc++.h>

#define bs bitset
#define mm(cont, val) memset(cont, val, sizeof cont);
#define rep(i, s, t) for (int (i) = int(s); (i) <= (t); (i) ++ )

using namespace std;

typedef vector<int> vi;
typedef stack<int> si;

int n;
bs<1000100> vis;

void workshi() {
    scanf("%d", &n);
	
	int s = n;
	si stk;
	
	rep(i, 1, n) {
		int t;
		scanf("%d", &t);
		if (t == s) {
			printf("%d ", t);
			vis[t] = true;
			if (s) while (vis[--s]);
			
			if (!stk.empty()) {
				int x = stk.top();
				while (x > s && vis[x]) {
					printf("%d ", x);
					stk.pop();
					if (!stk.empty()) x = stk.top();
                    else break;
				}	
			}
		} else {
			stk.push(t);
			vis[t] = true;
		}
	}
}

int main() {
	workshi();
	
	return 0;
}
